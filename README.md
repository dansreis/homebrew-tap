# homebrew-tap

Personal [Homebrew](https://brew.sh) tap for [Daniel Reis](https://github.com/dansreis)'s apps.

## SpecLens

A desktop reader for [OpenSpec](https://github.com/Fission-AI/OpenSpec) projects. macOS on Apple Silicon.

```sh
brew install --cask dansreis/tap/speclens
```

Builds are signed and notarized. See the [SpecLens repository](https://github.com/dansreis/speclens) for everything else.
